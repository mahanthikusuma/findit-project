# FindIT – Lost & Found Web Application
 A Smart Platform to Report, Search & Recover Lost Items
 
# Overview
FindIT is a full-stack MERN web application designed to simplify the process of reporting and recovering lost items.
It provides a centralized platform where users can post lost or found items, search listings, and claim ownership securely.

# Installation & Setup

# Clone Repository
git clone https://github.com/mahanthikusuma/findit-project
cd findit
# Install Dependencies

# Backend
cd backend
npm install
# Frontend
cd my-app
npm install

# Run Application
Backend (Port 5001)
npm run dev
Frontend
npm start
# API
Base URL: http://localhost:5001

# Notification System
Stores notifications in database
Triggered on claim requests & updates
Displayed in UI

# System Workflow
User posts lost/found item
Data stored in database
Users search items
Claim request is sent
Owner verifies claim
Notification is generated
